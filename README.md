# 🛒 Flipkart Product Scraper Dashboard

## 📌 Project Overview

The Flipkart Product Scraper Dashboard is a web application built using Python and Streamlit that allows users to search for products on Flipkart and scrape product information from multiple pages. The application displays the scraped data in an interactive dashboard and enables users to download the results as a CSV file.

## ✨ Features

* Search products by keyword
* Select the number of pages to scrape
* Extract product details from Flipkart
* Display data in an interactive table
* Download scraped data as a CSV file
* Simple and user-friendly interface

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Requests
* BeautifulSoup (bs4)
* lxml

## 📊 Workflow

1. Enter the product name.
2. Select the number of pages to scrape.
3. Click on **Start Scraping**.
4. View the scraped product data.
5. Download the data as a CSV file.
